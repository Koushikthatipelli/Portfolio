PROJECT OVERVIEW

One Workspace.
Total Productivity.

OneDesk brings together project management, task tracking, note taking and personal productivity into one modern workspace, eliminating the need for multiple applications.

My Role

Designed and developed the complete full stack application including UI, backend APIs, authentication, MongoDB integration and deployment.

Duration

Built over six weeks following an agile development workflow with continuous improvements and testing.

Platform

Responsive productivity platform featuring task management, projects, notes and personal dashboard.

Security

Secure JWT authentication, encrypted passwords, protected REST APIs and MongoDB cloud database.

  • Inefficient daily workflow
THE SOLUTION

TECH STACK

Built With Modern Technologies

OneDesk is developed using the MERN Stack, providing a scalable, secure and responsive productivity platform.

Frontend

  • HTML5
  • CSS3
  • JavaScript ES6
  • Responsive UI

Backend

  • Node.js
  • Express.js
  • REST APIs
  • JWT Authentication

Database

  • MongoDB Atlas
  • Mongoose
  • User Data
  • Cloud Storage

Deployment

  • Render
  • Vercel
  • GitHub
  • Cloud Hosting
