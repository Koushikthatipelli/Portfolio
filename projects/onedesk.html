PROJECT OVERVIEW

One Dashboard.
Unlimited Productivity.

OneDesk is a personal productivity platform built to simplify everyday work. It combines task management, notes, reminders and project organization into one clean and intuitive workspace.

MY ROLE

Full Stack Developer

Designed the complete user experience, developed frontend, backend APIs and integrated MongoDB authentication.

PROJECT TYPE

Productivity Platform

A modern workspace application helping users manage daily productivity efficiently from a single dashboard.

TARGET USERS

Students & Professionals

Designed for developers, students, freelancers and professionals looking for an organized digital workspace.
